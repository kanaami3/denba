## ⬜ 管理者にお願いする設定

### 1. 在庫追跡ONへの入れ替え（再インポート）
1. 商品管理 → インポート
2. 「ShopifyフォーマットのCSVファイルをアップロードする」→ 次へ
3. `shopify_products_import_tracking.csv` を選択
4. **「同じハンドルの既存商品を上書きする」を必ずON**
5. アップロード → インポート
- 結果：全42バリエーションが「在庫追跡ON／在庫0／売り切れ時は販売停止」になる

### 2. 再入荷通知アプリの導入
1. アプリ → アプリを追加（Shopify App Store）
2. 「Back in Stock」「再入荷通知」で検索し、**無料プランがある**アプリを選択
3. インストール → 売り切れ商品に「再入荷通知を受け取る」ボタンを表示する設定を有効化
- 運用：在庫を0→1以上に更新すると、登録者へ自動で通知メール

### 3. 配送設定（大阪発送・全国一律送料無料）
1. 設定 → ロケーション：発送元を**大阪の住所**に設定
2. 設定 → 配送と配達 → 配送プロファイル「料金を管理」
3. 日本国内の料金を **¥0（送料無料）** に設定
4. 北海道・沖縄・離島など別ゾーンがあれば、すべて¥0に
5. 保存

### 4. 決済設定（重要：特商法と一致させる）
1. 設定 → 決済 で、実際に有効化する決済手段を確定
2. **固定ページ「特定商取引法に基づく表記」のお支払方法を、実際に使える手段だけに修正**
   - 現状の記載：クレジットカード／代金引換／後払い／Apple Pay／コンビニ決済
   - Shopifyで未対応の手段があれば削除、対応手段のみ残す

### 5. テーマ（モノトーン・ミニマル）
- カスタマイズ → 色：背景#FFFFFF／文字#0A0A0A／ボタン背景#0A0A0A・文字#FFFFFF／セクション背景#F5F5F5／アクセント#9A7B34（任意）
- フォント：游ゴシック／ヒラギノ角ゴ／Helvetica系
- ボタン角丸を小さめ（0〜4px）に
- トップページのコピーは `shopify_copy_and_pages.md` の A章を反映
